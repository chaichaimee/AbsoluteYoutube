# AbsoluteYoutube

Мощный загрузчик YouTube для пользователей NVDA

![NVDA Logo](https://www.nvaccess.org/files/nvda/documentation/userGuide/images/nvda.ico)

**Автор:** chai chaimee  
**GitHub:** https://github.com/chaichaimee/AbsoluteYoutube

## Описание

AbsoluteYoutube — продвинутый аддон NVDA, позволяющий загружать видео и медиа с YouTube прямо из браузера в форматах MP3, MP4 или WAV. Включает интеллектуальные фоновые системы для эффективных и возобновляемых загрузок, обрезки видео, снимков, копирования коротких URL и полного менеджера неудачных загрузок — всё доступно простыми жестами и меню.

## Горячие клавиши

- **NVDA+Y** – Жест загрузки (определение тапов)  
  - Один тап: Загрузить как MP3  
  - Двойной тап: Загрузить как MP4  
  - Тройной тап: Загрузить как WAV  

- **CTRL+Shift+Y** – Контекстное меню / Папка  
  - Один тап: Открыть контекстное меню  
  - Двойной тап: Открыть папку назначения  

- **NVDA+Shift+Y** : Переключить режим плейлиста  

- **ALT+Windows+Y** : Циклически менять качество MP3 (128 → 192 → 256 → 320 kbps)  

Все комбинации используют определение тапов (~0.4 секунды). Переназначьте в NVDA → Жесты ввода.

## Функции

- **Мультиформатная загрузка (MP3 / MP4 / WAV)**  
  Нажмите NVDA+Y один, два или три раза для загрузки текущего видео в нужном формате. Поддерживает одиночные видео и плейлисты. В режиме плейлиста создаётся подпапка с именем плейлиста — одиночные файлы остаются отдельно.

- **Интеллектуальная фоновая система загрузки**  
  - Менеджер очереди: Загрузки по одной или с ограниченной параллельностью (до 4).  
  - Возобновление после перезапуска: Прерванные загрузки сохраняются и возобновляются автоматически или по запросу.  
  - Авторемонт: Очищает повреждённые временные файлы.  
  - Пропуск существующих: Автоматически пропускает уже загруженные файлы.  
  - Многопоточная загрузка: Разделяет на до 16 частей для ускорения (опционально).  
  Все функции в Настройках NVDA → AbsoluteYoutube.

- **Обрезка клипов видео**  
  На странице YouTube → Контекстное меню (CTRL+Shift+Y один тап) → Настройки обрезки.  
  Установить начало/конец → Выбрать MP3 (128–320 kbps), MP4 (H.265) или WAV → Предпросмотр → Загрузить клип.  
  Сохраняется как "Trimmed Clip 1.mp3" и т.д.

- **Снимок экрана**  
  Контекстное меню → Snapshot.  
  Загружает миниатюру высокого качества как .jpg полный.

- **Копирование короткой URL**  
  Контекстное меню → Copy video Shorten URL.  
  Преобразует ссылку в короткий формат и копирует мгновенно.

- **Менеджер неудачных загрузок**  
  Контекстное меню → Download fail manager.  
  Список с названием, длительностью, URL.  
  Правый клик:  
  - Удалить выбранное  
  - Загрузить сейчас  
  - Загрузить все  
  - Очистить всё  
  Неудачи сохраняются — повторите когда угодно.

- **Обновление yt-dlp**  
  Ручное: Настройки → Update yt-dlp now.  
  Авто: Включить "Auto-update yt-dlp on startup".

**Примечание**  
Все функции настраиваются в Настройках NVDA → AbsoluteYoutube. Горячие клавиши меняются в Жестах ввода.
